Узнать о локальных и удаленных ветках

Давайте посмотрим на ветки, доступные в нашем клонированном репозитории.
Выполните:

git branch

Результат:

$ git branch
* master

Как мы видим, в списке только ветка master.
Где ветка style? Команда git branch выводит только список 
локальных веток по умолчанию.


Список удаленных веток

Для того, чтобы увидеть все ветки, попробуйте следующую команду:
Выполните:

git branch -a

Результат:

$ git branch -a
* master
  remotes/origin/HEAD -> origin/master
  remotes/origin/style
  remotes/origin/master

Git выводит все коммиты в оригинальный репозиторий, 
но ветки в удаленном репозитории не рассматриваются как локальные. 
Если мы хотим иметь собственную ветку style, мы должны сами ее создать.
